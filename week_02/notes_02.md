# Week 02 - Functions, Lists, Dictionaries 📚

## Date
May 2026

## What I Learned

### Functions
- How to define functions using def
- Parameters - passing values into functions
- Return - getting values back from functions
- Default parameters
- Docstrings to document what a function does

### Lists
- Creating and accessing lists by index
- Methods: append(), remove(), sort(), len()
- Looping through lists with for loop
- Filtering items using conditions
- List comprehension - one liner version

### Dictionaries
- Key value pairs - like a real dictionary
- Accessing values by key
- Adding, updating, removing keys
- Looping using .items(), .keys(), .values()
- Checking if key exists using "in"
- Finding max, min, average from dictionary values
